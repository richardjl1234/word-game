🎙️ L1 单词音色对比

📌 用法：对每个 L1 单词，并排听三个版本：
  • FROM GAME：当前游戏里实际播放的 mp3（pitch=+2, 第一次批量生成）
  • PITCH +2：用相同参数（LovelyGirl, speed=0.85, pitch=+2）刚才新生成的
  • PITCH +5：用 pitch=+5 重新生成（高音版）
🎯 目标：对比每行三个 audio，应该是同一种声音（音调高低不同但音色一致）。 如果 FROM GAME 和 PITCH +2 听起来差距很大 → 游戏里 rob.mp3 实际生成时用了不同的参数。 如果听起来一样 → 之前用户感觉的"不一样"可能是不同上下文（句子 vs 单词）造成的错觉。

💡 判断标准：
  • 如果 FROM GAME ≈ PITCH +2 但 ≠ PITCH +5 → 游戏是正确参数，听感差异是 MiniMax 非确定性导致的小波动（可接受）
  • 如果 FROM GAME ≈ PITCH +5 → 之前 batch 实际用了 pitch=+5（不是 +2）
  • 如果三个都明显不同 → 可能是别的变量（speed, voice_id 等）
